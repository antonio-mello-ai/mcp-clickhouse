# CLAUDE.md — MCP ClickHouse

## Projeto

MCP server para interacao com databases ClickHouse. Construido com FastMCP.

## Stack

Python | FastMCP | clickhouse-connect

## Status

Ativo. Ferramenta interna para queries e operacoes ClickHouse.

## Setup obrigatório após clone

Executar uma vez: `git config core.hooksPath .githooks`

Isso ativa o pre-push hook que exige bump do arquivo `VERSION` quando há mudanças de código.
